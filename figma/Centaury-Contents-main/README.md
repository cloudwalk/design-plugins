## ğŸš€ Getting Started

### For Users

1. **Install the Plugin**
   - Open Figma Desktop
   - Go to Menu > Plugins > Browse Plugins
   - Search for "Centaury Contents"
   - Click "Install"

2. **Using the Plugin**
   - Select any text layer in your design
   - Right-click > Plugins > Centaury Contents
   - Choose your content type
   - Apply changes

### For Developers

1. **Prerequisites**
   - Node.js (v14 or higher)
   - Yarn or npm
   - Figma Desktop App

2. **Installation**
   ```bash
   # Clone the repository
   git clone https://github.com/your-username/centaury-contents.git
   cd centaury-contents

   # Install dependencies
   yarn install

   # Start development server
   yarn dev
   ```

3. **Development**
   ```bash
   # Run tests
   yarn test

   # Build for production
   yarn build
   ```
